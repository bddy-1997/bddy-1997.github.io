黑苹果安装Big Sur

配置

CPU: 英特尔I5-10600K
主板: 技嘉Z490 Gaming X AX
内存: 威刚DDR4 3200 8X2
硬盘: 铠侠RD 500G
显卡: 迪兰RX580 2304SP版本（2048SP无法驱动）
网卡: Fenvi T919

安装步骤

请务必阅读opencore官方文档 https://dortania.github.io/OpenCore-Install-Guide/
所有操作全部基于windows 10完成

  1. 鼠标移动到左侧USB Creation，选择Creation the usb目录下的 in windows，选择右侧的 macrecovery.py选项进入下载页面，进入下载页面选择0.7.6版本，选择OpenCore-0.7.6-RELEASE.zip进行下载。

2.解压缩得到所有文件，打开文件夹你会看到以下文件夹，打开Utilities文件夹，选择macrecovery并进入在文件夹中安装键盘shift键并单击鼠标右键选择在此处打开Powershell窗口选项，粘贴以下命令下载big sur恢复程序，下载完成后会得到两个新的文件。

3.将准备好的U盘进行格式化。

​ 6.在U盘根目录创建一个名为com.apple.recovery.boot的文件夹，将上面得到的两个文件拷贝到此文件夹。

​ 7.回到opencore0.7.6文件夹下打开X64文件夹后将EFI文件拷贝到U盘根目录。

​ 8.再次回到opencore0.7.6文件夹下将Docs文件夹下的Sample.list文件拷贝到EFI文件夹中的OC目录下并改名为config.list

​ 9.接下来就是对EFI进行修改，进入到OC文件夹中，从ACPI文件开始，进入

中关于ACPI的页面找到ACPI。是什么平台，按需选择就行。新手就不建议自己编译SSDT文件了，等你都弄明白了，再回过头来自己编译更适合你电脑的SSDT。

ACPI

Drives

这个文件夹中本身就包含了两个必须添加的驱动，否则启动会失败剩下的删除就行了。

Kexts

增加你主板的一些驱动程序，如网卡，声卡。CPU温度，USB等。

Tools

一般只用这一个文件就行了.

这样基本上驱动就添加完成了，接下来是修改config.plist文件。你需要下载两个文件

1.ProperTree-master 这是用来打开并修改config文件的工具。

2.GenSMBIOS-master 这是生成机器信息的工具。

将下载好的两个文件文件解压缩，选择ProperTree批命令文件并打开。

这一步相对复杂我也不方便录制gif文件，就以截图的形式查看吧。

将config调整完并保存也就大功告成了。接下开就是修改BIOS环节，由于接下来步骤无法录制，也就不放出来了，照猫画虎吧。修改完BIOS保存并退出，电脑自动重启，选择U盘选项，开始跑码，直到进入系统恢复页面基本上就离开机八九不离十了。格式化硬盘，选择APSF分区模式，然后就可以在线安装系统了。等进度条走完自己会重启2-3次，直到它自己进入系统。

修改BIOS设置

仅针对本次使用的主板
禁用

  • Fast Boot
  • Secure Boot
  • Serial/COM Port
  • VT-d (can be enabled if you set DisableIoMapper to YES)
  • CSM
  • Intel SGX
  • Intel Platform Trust
  • CFG Lock (MSR 0xE2 write protection)(This must be off, if you can’t find the option then enable AppleXcpmCfgLock under Kernel -> Quirks. Your hack will not boot with CFG-Lock enabled)

启用

  • VT-x
  • Above 4G decoding
  • EHCI/XHCI Hand-off
  • OS type: Windows 8.1/10 UEFI Mode
  • SATA Mode: AHCI

接下来就是将U盘的EFI文件拷贝到电脑硬盘，因为目前是用U盘引导进入系统的。

拷贝完成将U盘拔下重启电脑看看不用U盘是否能够进入系统，成功进入系统意味着黑苹果已经安装完成了。对于定制声卡，USB就不在本次范畴也就不进行演示了。
